# Simple Python Chatbot

*Description*

This is a basic rule-based chatbot built in Python. It takes user input, matches it with predefined responses, and prints a reply. If the input is not recognized, it shows a default message.

*How It Works*

The program runs in a loop, takes user input, converts it to lowercase, and checks it against a dictionary of responses. Type "exit" to stop the program.
Supported Inputs
hello → Hi!
goodbye → Bye!
good morning → Good Morning!
good afternoon → Good Afternoon!
good night → Good Night!
If not recognized: I don't understand
Type exit to stop the program

*Features*

Simple terminal chatbot
Dictionary-based responses
Beginner-friendly Python project
